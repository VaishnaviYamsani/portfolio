COLORORA

Colorora is a web extension which allow users who suffer with color deficiency to interact with the pages in a better way.You can add this extension to the browser and then it changes the color of the page you are surfing according to the type of disesase the user has selected. The diseases are provided in the drodown menu in the extension.
